# Project Context: Kalvis Lejasmeiers — Architect Portfolio

## Project Overview

Personal portfolio website for **Kalvis Lejasmeiers**, an architect with 10 years of experience based in **Liepāja, Latvia**.

## Core Purpose

- Establish credibility as an independent architect
- Make it easy for potential clients to understand services and contact
- Create a personal, human feel (not corporate)

## Key Requirements

- Pure HTML + CSS + minimal vanilla JS (no frameworks)
- Mobile responsive
- Fast loading
- Domain: **kalvisl.lv**
- Design reference: David Chipperfield website (https://www.davidchipperfield.com)

## Design Philosophy

- **White space IS the design**
- Images carry all visual weight
- Typography is invisible infrastructure
- No gradients, shadows, rounded corners, icons, or animations (except subtle opacity fade)

## Color Palette

- Background: Pure white (#ffffff)
- Text: Black (#111111)
- No accent colors

## Typography

- Font: 'Helvetica Neue', Helvetica, Arial (clean sans-serif)
- Navigation: all lowercase
- Project metadata: small, precise label text

## Site Structure (Multi-Page with Single-Page Home)

**Homepage (index.html) - Single Page Scroll:**

1. Navigation (fixed top)
2. Featured Projects (3 projects)
3. All Work text list (7 projects)
4. Pieredze Summary (new section with stats + timeline)
5. About section
6. Contact section

**Additional Pages:**

- `/projekti/` - Projects archive with filtering
- `/par-mani/` - About page with detailed biography
- `/pieredze/` - Professional experience page (new)
- `/kontakti/` - Contact page with form
